Role Name
=========

Ansible role to install and configure a rsnapshot.

Requirements
------------

This role requires Ansible 2.4 or higher, and platform requirements are listed in the metadata file.

Dependencies
------------

None.

Example Playbook
----------------
```
- hosts: servers
  become: true
  roles:
     - ansible-role-rsnapshot
```
